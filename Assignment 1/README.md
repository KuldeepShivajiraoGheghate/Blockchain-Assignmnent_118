# 📄 Smart Contract README

## 📌 Contract Purpose
This smart contract stores and retrieves a single integer value on the blockchain.  
It demonstrates basic Solidity concepts such as:
- State Variables  
- Functions  

---

## ⚙️ Functions and Logic

| Function | Description |
|----------|------------|
| `set(uint256 _data)` | Stores a value on the blockchain |
| `get()` | Returns the stored value |

---

## 🚀 Compilation and Deployment Steps

1. Open Remix IDE: https://remix.ethereum.org  
2. Create a new file named `contract.sol`  
3. Paste the smart contract code  
4. Compile using **Solidity Compiler version 0.8.x**  
5. Go to the **Deploy & Run Transactions** tab  
6. Select environment: **Browser Provider (MetaMask)**  
7. Connect your MetaMask wallet  
8. Click **Deploy** and confirm the transaction  

---

## ⭐ Notes
- Ensure MetaMask is installed in your browser  
- Select the correct network before deployment (e.g., Sepolia Testnet)  
- Keep testnet ETH for gas fees
