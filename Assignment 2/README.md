## 📌 Objective
To deploy a Solidity smart contract on the **Polygon Amoy Testnet** using Remix IDE and MetaMask.

---

## 📂 Files Included
- `contract.sol` → Smart contract file  
- `README.md` → Documentation of deployment process  
- `Screenshots/` → Proof of deployment and transaction  

---

## ⚙️ Tools Used
- Remix IDE  
- MetaMask Wallet  
- Polygon Amoy Testnet

## 🚀 Deployment Steps

### 🔹 Step 1: Open Remix IDE
1. Go to https://remix.ethereum.org  
2. Create a new file `contract.sol`  
3. Paste the smart contract code  

---

### 🔹 Step 2: Compile Contract
1. Go to **Solidity Compiler** tab  
2. Select version **0.8.x**  
3. Click **Compile**  

---

### 🔹 Step 3: Setup MetaMask
1. Install MetaMask extension  
2. Add **Polygon Amoy Network** using the details above  
3. Switch to Polygon Amoy network  

---

### 🔹 Step 4: Connect MetaMask to Remix
1. Go to **Deploy & Run Transactions**  
2. Select Environment → **Injected Provider - MetaMask**  
3. Connect your wallet  

---

### 🔹 Step 5: Deploy Contract
1. Click **Deploy**  
2. Confirm transaction in MetaMask  
3. Wait for successful deployment  

---

## ⭐ Notes
- Ensure you have testnet POL tokens for gas fees  
- Double-check network selection before deploying  
- Use Polygonscan to verify your contract
